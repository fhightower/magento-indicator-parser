Esta nova versão introduz novas funcionalidades e torna o ISPConfig 3, uma alternativa completa para as soluções DNS, E-mail, FTP e Sites . ISPConfig é capaz de alterar as configurações do sistema, gerenciar as contas de e-mail e ftp em seu servidor, monitorar o desempenho do servidor, adicionar e gerenciar sites, gerir zonas de DNS e muito mais.

 ISPConfig é também uma ferramenta para o gerenciamento de seus clientes no servidor. Se o plano é criar um Servidor para hospedar as páginas que você criar para os seus clientes, o ISPConfig é uma boa opção como gerenciar os clientes e como dar-lhes um painel de controle simples de gerir as suas necessidades.

O melhor em ISPConfig é que pode haver diferentes papéis de utilizador. O administrador é capaz de usar todos os recursos do ISPConfig. Os clientes poderão gerenciar apenas a sua conta  sites, e-mails e contas FTP que eles usam. Há também um outro papel o revendedor. O revendedor é uma pessoa que é capaz de criar outros clientes e seus domínios, mas não é capaz de mudar as configurações do servidor.

Software Profissional para Provedores, poderá cadastrar Revendedores e até fazer Alta Disponibilidade de Servidores.

  GLPI - Inventário e Help Desk

 

Site Oficial: http://www.glpi-project.org


GLPI é um Sistema de Gestão de Ativos de TI e que permite aos administradores de TI gerenciar a manutenção, controle e manutenção de TI inventário. Gerenciar e rastrear ativos como computadores, software, periféricos e consumíveis, e sua manutenção associados, documentação e solicitações de Suporte.

GLPI tem gestão de ativos, permitindo a entrada fácil de seus dados de inventário. Modelos no GLPI permitem que você preencher os campos de informação e até mesmo pré-especificar valores de campo de formulário. Por exemplo, o "Monitores" modelo pode inserir automaticamente os detalhes de contato para o responsável técnico de cada vez que você adicionar um novo monitor.

Características adicionais incluem relatórios GLPI capacidades que permitem revisão rápida da história de ativos de manutenção e procedimentos, ajuda a funcionalidade de rastreamento com mesa de trabalho e de roteamento, e um sistema de solicitação que permite aos usuários solicitar diretamente suporte de help desk.

Plugins disponíveis GLPI permitem que você adicionar ainda mais funcionalidade, como geração de código de barras, listas de verificação de criação, gestão de sala e muito mais. Com uma interface web-based administrativa, GLPI permite o uso multi-usuário, multi-local, e inclui um sistema de notificação de correio que entrega os pedidos de help desk directamente aos fornecedores de manutenção.

GLPI - Características e vantagens

Controle de estoque - Construir um banco de dados simples, em seguida, gerenciar e controlar computadores, incluído software, periféricos e consumíveis associados. Incluem informações valiosas, tais como locais de ativos, licenças e custos.

Gestão de inventário - Acompanhar e gerenciar solicitações de apoio mapeados diretamente para os ativos atendidos. Além disso gerenciar documentos relacionados com o seu inventário de TI, incluindo FAQ, e contratos com fornecedores.

Pedido de suporte e encaminhamento - Serve usuários finais de maneira mais eficiente. GLPI permite que os usuários ativos para solicitação de serviços diretamente através da web. Administradores de TI podem receber notificações de solicitação automática, encaminhar pedidos aos técnicos apropriados e status bilhete pista.

Geração de Relatórios - Gera relatórios associados a seus ativos, incluindo técnicos designados e histórico de manutenção, em seguida, exportar seus relatórios em PDF, CSV, SLK, ou no formato XML.

Suporte multi-usuário - Não deixe seus trabalhadores remotos ou locais para fora no frio, GLPI permite a coordenação, acompanhamento, suporte e manutenção de ativos em locais fora do local.

Pesquisa - GLPI permite-lhe procurar rapidamente o status, a história, e manutenção de seus ativos.

Campos personalizáveis ​​- Personalize campos de exibição para atender às suas necessidades de informação único ativo.

Notificações - É hora de fim de toner? Receber uma notificação automática de seus ativos eventos associados, tais como níveis de estoque de consumíveis, e expiração de contratos futuros e licenças.

Estatísticas - Gerar relatórios estatísticos sobre suas atividades de gestão de ativos, incluindo relatórios de hardware, localização do usuário, categoria, prioridade, ou contrato.

Multi-lingual - Mais de 22 idiomas são suportados.

Site Oficial:  http://www.cacti.net

Cacti é uma solução gráfica completa para rede projetada para aproveitar o poder do RRDTool 's de armazenamento de dados e funcionalidades gráficas. Cacti proporciona um rápido poller, avançado gráfico templating, vários métodos de aquisição de dados, e gerenciamento de usuários funcionalidades out of the box. Tudo isso é envolto em uma interface intuitiva, fácil de usar que faz sentido para LAN-sized instalações até redes complexas com centenas de dispositivos.

O que é Cacti?

Cacti é uma interface completa para RRDTool, que armazena todas as informações necessárias para criar gráficos e preenchê-los com dados em um banco de dados MySQL. O frontend é completamente PHP conduzido. Além de ser capaz de manter Gráficos, Fontes de Dados e Arquivos Round Robin em um banco de dados, cactos lida com a coleta de dados. Também há suporte SNMP para aqueles acostumados a criar gráficos de tráfego com MRTG.

Fontes de dados 
Para lidar com a coleta de dados, você pode alimentar cactos os caminhos para qualquer script externo / comando, juntamente com todos os dados que o usuário terá que "preencher", cactos, então, reunir esses dados em um cron-job e preencher um banco de dados MySQL / a round robin arquivos.

Fontes de dados também podem ser criados, que correspondem aos dados reais no gráfico. Por exemplo, se um usuário deseja representar graficamente os tempos de ping para um host, você pode criar uma fonte de dados utilizando um script que pings um host e retorna seu valor em milissegundos. Após definir as opções para RRDTool como a forma de armazenar os dados, você será capaz de definir quaisquer informações adicionais que a fonte de entrada de dados requer, como um host para o ping neste caso. Uma vez que uma fonte de dados é criado, ele é automaticamente mantida em intervalos de 5 minutos.

Gráficos 
Uma vez que uma ou mais fontes de dados são definidos, um gráfico RRDTool pode ser criado usando os dados. Cacti permite criar quase qualquer gráfico RRDTool imagináveis ​​usando todos os tipos de RRDTool padrão gráfico e funções de consolidação. A área de seleção de cor ea função de preenchimento automático de texto também ajuda na criação de gráficos para tornar o processo mais fácil.

Não apenas você pode criar gráficos com base em RRDTool cactos, mas há muitas maneiras de exibi-los. Juntamente com uma "exibição de lista" padrão e um "modo de visualização", que lembra o 14all frontend RRDTool, há uma "árvore", que permite que você coloque os gráficos em uma árvore hierárquica para fins de organização.

Gerenciamento de usuários 
Devido as muitas funções de cactos, um usuário ferramenta de gestão de base é construída em que você pode adicionar usuários e dar-lhes direitos a determinadas áreas dos cactos. Isso permitiria que alguém para criar alguns usuários que podem alterar os parâmetros do gráfico, enquanto outros só pode exibir gráficos. Cada usuário também mantém as suas próprias definições quando se trata de gráficos de visualização.

Templating 
Por último, cactos é capaz de escala para um grande número de fontes de dados e gráficos por meio do uso de modelos. Isso permite a criação de um único gráfico ou modelo de fonte de dados que define qualquer fonte de gráfico ou dados associados a ela. Modelos de host permitem que você defina as capacidades de um host para cactos pode poll-lo para informações sobre a adição de um novo hospedeiro.
