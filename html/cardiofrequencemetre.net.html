Tout savoir sur les CARDIOFREQUENCEMETRES en quelques lignes


Polar à une gamme de Gps pour le running le GPS M400, pour le triathlon (compteur de natation) ou le trail (altimètre barométrique) le GPS V800 et également pour le vélo, le GPS V650.
Ces modèles existent en 2 versions, une avec cardio ou sans le cardio.
Vous trouverez également des bracelets de suivi d'actiivité, le Polar Loop et des montres connectées les Polar A300
Suunto à une gamme de cardio fréquencemètre les Suunto Quest.
Vous trouverez également des GPS, les Suunto Ambit 3 Run ou les Ambit 3 Sport pour avoir le compteur de natation en plus ou bien les Ambit 3 Peak qui ont l'altimètre barométrique.
Ces modèles existent avec ou sans cardio.
Timex propose des montres chrono  homme ou  femme avec temps intermédiaires et intervales programmables.
Ils ont également un modèle GPS le Timex Run x20
Tomtom à une gamme de GPS, le Runner ou le multisport qui a le compteur de natation et qui est vendu avec un support vélo.
Tomtom propose également un GPS avec cardio mais sans la ceinture thoracique, la prise de pouls ce fait au poignet, le Tomtom Runner cardio.
Garmin propose le GPS Forerunner pour les coureurs ou triathlètes qui existent en version avec cardio les Forerunner Hrm et aussi en version Forerunner lady.
Les GPS Fenix 3 sont plus adaptés pour le trail.
Garmin a également une gamme de bracelets pour le suivi d'activité, les Garmin Vivovit.
