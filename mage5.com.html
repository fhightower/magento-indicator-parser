Why I need Mage5 In My Store

Mage5 framework help you edit your site inline and it is very flexible framework for your developers to make custom theme, block types and connecting admin package with Mage5 Block types. More over it reduces the magento development skinning time. Once your site is live, site owners can easily change images/contents (CMS) very easily using the powerful Mage5 inline editing framework.

Features of Mage5

Mage5 is easy for site owners to use, flexible for developers to work with, and ready for Community/Go and Enterprise Ready.

Some of many features include:

  • WYSIWYG text editor
  • In-context editing toolbar – change your website as you surf!
  • Automatic navigations – add a page anywhere and it's updated everywhere
  • File manager with bulk upload
  • Drag-n-Drop content around your pages
  • An open marketplace of add-ons and themes
  • Out of the box includes a photo slide show, You Tube integration, Flash support, search, RSS, secure file distribution,Google Maps, and a form generation tool
